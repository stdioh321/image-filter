Serviços utilizados
	* Pho.to API - http://developers.pho.to/
		- Aplica os filtros e efeitos nas imagens;
		- Limite de 100 filtros por dia;
		- Atualmente utilizando TOKEN do site: "http://art.pho.to/"
	
	* Imgur API - https://apidocs.imgur.com/?version=latest
		- Armazena e publica a imagem  para ser acessada pelo pho.to api.
		- Limite de 1250 uploads por dia.
	
	* Cloudimage API - https://www.cloudimage.io/
		- Recebe e deixa publica a imagem filtrada do pho.to api.
		- Limite de 25GB de trafego mensal
