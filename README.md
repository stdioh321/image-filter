# Image Filters

####[Demo Page](https://image-filters.netlify.com/)

## Run locally
```
$ git clone https://github.com/stdioh321/image-filter.git
$ cd image-filters
$ cd client
$ npm install
$ ng serve
```
## Browser
http://localhost:4200

### Requirements
1. [Node/NPM](https://nodejs.org/en/)
2. [Angular CLI](https://cli.angular.io/)
